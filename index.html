Deskripsi Sistem

Sistem Penjualan Alat Tulis Sekolah Berbasis Website merupakan sistem yang dirancang untuk membantu proses penjualan alat tulis secara digital. Sistem memungkinkan admin mengelola produk, memantau stok barang, mengelola transaksi, serta menghasilkan laporan penjualan secara efektif.

Konsep OOP

Class & Object

User, Admin, Pelanggan, Produk, Transaksi, dan Pembayaran.

Inheritance

Admin dan Pelanggan merupakan turunan dari User.

Encapsulation

Data produk seperti harga dan stok dikelola melalui class Produk.

Polymorphism

Method pembayaran dapat berbeda implementasi pada TransferBank dan EWallet.

Source Code

produk.dart

class Produk {
  String kodeProduk;
  String namaProduk;
  double harga;
  int stok;

  Produk(this.kodeProduk, this.namaProduk,
      this.harga, this.stok);

  void tampilkanProduk() {
    print("📦 Produk: $namaProduk");
    print("Harga: Rp${harga.toInt()}");
  }
}

pelanggan.dart

class Pelanggan {
  String nama;
  String alamat;

  Pelanggan(this.nama, this.alamat);

  void infoPelanggan() {
    print("👤 Nama: $nama");
  }
}

transaksi.dart

class Transaksi {
  String id;
  double total;

  Transaksi(this.id, this.total);

  void cetakStruk() {
    print("Total: Rp$total");
  }
}

inheritance.dart

class User {
  String nama;

  User(this.nama);
}

class Admin extends User {
  Admin(String nama) : super(nama);
}

polymorphism.dart

class Pembayaran {
  void bayar() {}
}

class TransferBank extends Pembayaran {
  @override
  void bayar() {
    print("Transfer Bank");
  }
}
